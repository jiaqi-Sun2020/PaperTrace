# Workspace contract

`.agents/adaptive-teach/` contains Mission, teaching settings, independent session directories, and regenerated `derived/` reports. Settings may hold language, lesson duration, practice ratio, HTML choice, review frequency, allowed local roots, Mission relevance, and explicitly supplied prerequisite maps. They must not copy profile concepts, events, sources, statuses, or queue entries.

Mission changes require a clear user confirmation and a corresponding session record. The directory is ignored by Git because it can contain personal diagnostics and answers.
