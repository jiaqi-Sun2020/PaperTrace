# Teaching cycle

Validate profile → build candidates → select one mode → optionally diagnose (1–3 prompts) → write one short lesson → collect actual performance → validate handoff → delegate import → write session record. Stop on invalid profile or invalid feedback; neither condition may alter the profile.

Use `diagnose` for insufficient evidence, `teach` for an explicit weak gap, `review` for a due known/mastered concept, `prerequisite` for a configured minimal blocker, and `transfer` only after an established concept needs a new-context check.
